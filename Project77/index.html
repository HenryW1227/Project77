To set up VSC (Visual Studio Code) You must create an empty folder, go to VSC and open the folder by pressing File, Open folder and select the empty folder.

Then press new file in VSC and name it " index.html " and then begin coding. You can code with HTML or JavaScript. For Js, use script in greater than signs.

When coding in JavaScript you can print anything you like on the console using " console.log() " and acsess the console by pressing Ctrl+Shift+I.

The console is used to test the output of your code and can be acsessed by the coder. And you can put anything you want onto the console.
